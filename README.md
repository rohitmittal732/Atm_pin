A simple Python ATM PIN verification program with limited attempts and invalid input handling. Ideal for beginners learning conditions and loops.
# ATM PIN Verification System

This is a simple Python project that simulates an ATM PIN verification system.  
The program asks the user to enter a 4-digit PIN and checks it against a stored PIN value.  
Users get limited attempts, and the program also handles invalid inputs such as letters 
or PINs that are not exactly 4 digits.

## Features
- 4-digit PIN verification
- Error handling for invalid input
- Limited number of attempts
- Simple and beginner-friendly logic
